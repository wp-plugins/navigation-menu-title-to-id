Contributors: Pascaline Chotard
Tags: navigation menu, id, attributes, title
Requires at least: 3.0
Tested up to: 3.0.1
Stable tag: trunk

This plugin allow you to use the title attribute of the a tags of your menu as ids.

== Description ==

For css ergonomic convenience, this plugin allow you to set an id attribute to your menu a tags

== Installation ==

1. Unzip navigation-menu-title-to-id.zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
